**Polymarket Account Analyzer** 是一款 **只读** 的 Rust 工具：输入任意 Polymarket 钱包地址，从公开 [Data API](https://data-api.polymarket.com)（及可选 Gamma、Goldsky 子图）拉取数据，生成结构化 **AnalyzeReport** JSON，并支持 **命令行** 与 **`GET /analyze/:wallet`**。不涉及下单、签名或私钥。

<Card title="快速开始" icon="rocket" href="/quickstart" horizontal>
  克隆仓库、编译、运行一次分析与本地 API。
</Card>

<Card title="服务器部署" icon="server" href="/deployment-server" horizontal>
  Docker 仅 Postgres、宿主机 Rust、域名与 HTTPS（如 api.analyzer.forevex.trade）。
</Card>

## 产品能力

<CardGroup cols={2}>
  <Card title="全市场类型" icon="chart-network">
    不限于 5 分钟市场：BTC/ETH 短线、1h、daily、事件、政治、体育等均可纳入统计。
  </Card>
  <Card title="多维报告" icon="chart-column">
    Lifetime、市场分布、价格分桶、UTC 活跃时段、交易模式、胜率与策略反推（`rule_json` + 伪代码）。
  </Card>
  <Card title="CLI + HTTP" icon="terminal">
    `analyze` 输出 stdout 或 `--out`；`serve` 提供 REST，便于前端或脚本集成。
  </Card>
  <Card title="可选 Postgres" icon="database">
    报告缓存、raw / canonical 落库、对账摘要、榜单 `GET /leaderboard`（需数据库）。
  </Card>
</CardGroup>

## 你能得到什么

| 能力 | 说明 |
| --- | --- |
| **分析引擎** | 基于 Data API `trades` 主路径；可选子图赎回 / fills / PnL 与用户仓位 |
| **对账与血缘** | v0/v1 对账、`quality_alert`、报告内 `provenance` / `data_lineage` |
| **报告模式** | 默认内存 Data API；`analytics.source = canonical` 或 `report-from-canonical-run`（PG 重放） |
| **输出形态** | `schema_version` 2.x JSON；含 `frontend` 块（大单、持仓、最近成交等）供 UI 使用 |

<Warning>
  **非交易产品**：不提供 CLOB 下单、Builder 归因或 Gasless Relayer。EOA / Safe 用户若要在链上交易，请使用 [Polymarket Builder 文档](https://docs.polymarket.com/builders/overview) 中的官方客户端与 Relayer 方案。
</Warning>

## 工作流程

<Steps>
  <Step title="提供钱包地址">
    CLI：`analyze 0x…`；HTTP：`GET /analyze/0x…`（路径参数勿对 `0x` 做多余编码）。
  </Step>
  <Step title="拉取公开数据">
    分页请求 Data API；按需叠加 Gamma 元数据、子图与可选 PG 持久化。
  </Step>
  <Step title="构建报告">
    聚合指标、模式检测、策略推断；写入缓存（若启用 Postgres）。
  </Step>
  <Step title="消费结果">
    CLI 打印或写文件；HTTP 返回 JSON；下游前端（如独立 Next 应用）可直接绑定字段。
  </Step>
</Steps>

## 上手步骤

<Steps>
  <Step title="获取源码">
    在 monorepo 中进入 **`polymarket-account-analyzer/`**（或克隆 [Polymarket-Analyzer](https://github.com/a9research/Polymarket-Analyzer) 仓库根目录）。
  </Step>
  <Step title="编译">
    `cargo build --release`，二进制名为 **`polymarket-account-analyzer`**。
  </Step>
  <Step title="跑一次分析">
    `cargo run --release -- analyze <wallet> --out report.json`（可选 `--config`、`--with-subgraph` 等）。
  </Step>
  <Step title="（可选）启动 API">
    `cargo run --release -- serve --bind 127.0.0.1:3000`，详见 [API 参考](/api-reference/introduction)。
  </Step>
</Steps>
